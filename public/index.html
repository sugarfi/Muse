MUSE

Code, Music, Collab

Have you ever wanted to write code and have it create music? Maybe you and your friends wanted to collaborate and create a project? Now you can! Muse is an interactive and collaborative Javascript IDE designed to make music. Create a room, share the link with your friends, write your code, then save it and share it for everyone to see! Creating music has never been easier! Ready to start? Simply create a room below!

Background

Muse was created for the 2020 Piemadd Hackathon. It was built using Tone.js, socket.io, Express, and Ace. The code was made entirely by me, sugarfi, and is licensed under the MIT license.
